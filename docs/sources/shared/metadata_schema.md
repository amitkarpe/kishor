# Metadata Schema

## Required fields

- `title`
- `category`
- `topic`
- `slug`
- `date`
- `description`
- `tags`
- `status`

## Optional fields

- `audience`
- `framework`
- `style_system`
- `source_type`
- `legacy_source`
- `notes_status`

## Example

```yaml
title: Proud Hindu Today
category: discussion
topic: Hindu identity and confidence
slug: proud-hindu-today
date: 2026-03-15
description: A discussion deck on Hindu identity, confidence, and visible dharmic habits.
tags:
  - hindu-identity
  - confidence
  - kishor
status: ready
audience: kishor-12-17
framework: Discussion_Framework_for_Kishor_Workflow
style_system: Kishor_Presentation_Style_System_Workflow
```
