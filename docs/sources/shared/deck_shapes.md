# Deck Shapes

## Default quiz deck shape

- title
- optional quick intro
- repeating quiz question rhythm
- answer reveal rhythm
- concise wrap-up

## Default discussion deck shape

- title
- hook question
- prompt / hint slide
- discussion slide
- repeated for each major question
- optional reflection slide
- final one-challenge slide

## Recommended slide-count guidance

These are guidelines, not rigid rules.

### Quiz
- 8 to 20 slides for short live use
- more only if the topic clearly needs it

### Discussion
- 8 to 16 slides for most sessions
- more only for deeper sessions with multiple questions
