# Quality Checklist

## Before creating a new deck

- correct category selected
- correct framework loaded
- correct style system loaded
- topic scope is sufficiently clear
- target slug is clean

## Before finalizing a deck

- `slides.md` exists
- `meta.yml` exists
- `notes.md` exists
- metadata is complete
- structure follows the framework
- description is homepage-friendly
- no unrelated deck was modified

## Before importing legacy material

- provenance is recorded
- import status is explicit
- normalization is documented
- missing content is clearly marked
