# Quiz Framework for Kishor - Workflow

## 1) Purpose

This source defines how ChatGPT should help Amit create **quiz-first Kishor sessions**.

This framework is only for:

- quiz workflow
- quiz presentation workflow
- downloadable quiz PPTX creation


This framework is **not** for:

- discussion sessions
- storytelling sessions
- value talks
- lecture-style presentations
- mixed quiz + discussion workflows

Do **not** mix this source with **Discussion Framework for Kishor — Workflow.md**.

## 7) Mandatory 2-Phase Workflow

Every quiz request using this framework must follow exactly 2 phases.

## Phase 1 — Plain Content Only
Before creating anything in apps/tools:

1. read this source
2. if slides are involved, also read **Kishor Presentation Style System - Workflow.md**
3. ask only the minimum clarification questions if needed
4. propose the quiz structure in plain text only
5. show sample content
6. stop and ask Amit to approve

Do **not** create @Quizzes or PPT in Phase 1.

## Phase 2 — Only After Amit Says "Go"
After Amit approves and says **go**:

1. create quiz using **@Quizzes**
2. create a downloadable **PPTX presentation deck**
3. use an appropriate filename based on the topic
4. ensure the deck remains quiz-first, not a normal informational presentation

---

## 8) Clarification Questions Rule

Ask only short questions if needed.

Recommended areas:

1. topic scope  
   - India or World?
   - exact category?

2. quantity  
   - 8 / 10 / 12 people or items?

3. difficulty  
   - easy / medium / mixed?

4. output  
   - plain content first?
   - then @Quizzes + PPT?

5. image mode  
   - photo mode
   - text-only mode

Do not ask broad vague questions.  
Do not ask more than needed.

## 10) Fast Zoom Lite Mode Structure

Use this structure:

```text
Title
Identify
Challenge
Answer
Identify
Challenge
Answer
Identify
Challenge
Answer
...
Shared Value Hook
Vote / Closing
```

This is the standard quiz deck structure for Kishor.

No divider slides between people.  
No timer by default.  
Start quickly after title.

---

## 11) Per-Person Slide Logic

Each person/item uses only 3 slides in Fast Zoom Lite Mode.

## Slide A — IDENTIFY
Goal:
- recognition
- excitement
- immediate engagement

Must include:
- one **large photo** of the person
- centered / prominent layout
- clean face-visible image
- hint shown on slide
- 4 options: A / B / C / D

## Slide B — CHALLENGE
Goal:
- reveal person
- immediately ask one harder follow-up question

## Slide C — ANSWER
Goal:
- lock learning quickly
- show only the correct answer
- avoid repetition

---

## 12) Identify Slide Rule

This is mandatory unless Amit explicitly asks for text-only mode.

Every Identify slide must include:

- one large clear photo of the person
- face visible
- clean background
- single person only
- no collage
- no clutter
- no crowded montage

Preferred layout:

```text
[Large centered photo]

Who is this?
Hint: ______
A / B / C / D
```

This is the default layout standard.

---

## 13) Hint Rule

For Item 1 / Identify slide:

- hint must be shown on the slide
- hint must be short
- ideal length = 4–6 words
- hint should improve recognition, not reveal too much

Example:
- Master Blaster
- Flying Sikh
- Olympic Javelin Gold
- Chess World Champion

---

## 14) Wrong Options Rule

Wrong options must be:

- plausible
- same category
- same domain
- same era if possible

Examples:

If the correct answer is **Sachin Tendulkar**, better wrong options are:
- Rahul Dravid
- Brian Lara
- Ricky Ponting

Avoid unrelated wrong options.

Do not use random cross-category distractors unless Amit explicitly wants mixed-category confusion.

---

## 15) Challenge Slide Rule

Reveal slide must follow this format:

```text
Correct Answer: <Name>

Next Question:
<Harder follow-up question>
```

The harder question should be:

- one line if possible
- clear
- not tricky
- based on achievement / year / award / record / title / country / company / invention / field

Examples:
- Which Olympic Games did he win gold?
- Which company did he build?
- Which award did she win?
- What was he famous for?

---

## 16) Answer Slide Rule

The answer slide must be simple.

Use only:

**Title:**  
Answer Check

**Main text:**  
Answer: <KEY ANSWER>

Rules:
- highlight the key answer in orange
- do not repeat the full question again
- do not add a confirm question like:
  - True or False?
  - Did he win in Tokyo 2020?
- keep it short and clean

This is intentional to avoid repetition and make slides lighter.

---

## 17) Shared Value Hook Rule

Do **not** repeat value hook for every person in Fast Zoom Lite Mode.

Instead, add one **shared value hook** near the end of the deck.

Example:

```text
Why do great people stand out?

Practice
Focus
Discipline
Humility
```

Use orange highlight for keywords if style source recommends it.

---

## 18) Reflection Prompt Rule

Do **not** repeat the reflection prompt on every person slide.

Instead, include one shared reflection prompt near the end.

Preferred placement:
- second-last slide
- or last content slide before closing

Example:
- What habit made them great?
- What can we copy from them this week?
- Which quality inspired you most?

This prompt may appear on screen.

---

## 19) Closing Slide Rule

End the deck with a light closing interaction.

Preferred default:

```text
Vote:
Who inspired you most?
```

This works better than a long closing message for live Kishor Zoom sessions.

---

## 20) Slide Density Rule

Keep quiz slides minimal.

For quiz slides:

- max 2 ideas per slide
- no paragraphs
- no lecture text
- no dense explanations

Example:
- title
- image
- question
- options

That is enough.

---
